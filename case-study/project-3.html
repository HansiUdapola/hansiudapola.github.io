Strategic Partnership Evaluation Dashboard

A complete redesign and development of a web-based psychometric evaluation tool used to monitor and assess the health of ongoing strategic initiatives. Engineered to provide accurate, comparative feedback between partnering organisations across six core strategic pillars.

Software Developer

Full-Stack Development · UX Design · Serverless Architecture · Database Modelling

Outcome
Redesigned and engineered a web-based psychometric evaluation tool.
Web Development React & Vite NoSQL Database Data Visualisation UX/UI Design

The Challenge (Evaluating Partnership Health)

Evaluating the health of a strategic bilateral partnership requires nuance, accuracy, and clear perspectives from both sides. The previous iteration of this evaluation application suffered from structural and experiential limitations that hindered meaningful data collection:

Pain Point 01
Disconnected Data

Specific active projects were not accurately mapped to their overarching strategic pillars, making it difficult to extract relevant, context-driven feedback.

Pain Point 02
Lack of Comparison

There was no visual mechanism to directly compare the perspectives of the two partnering organisations on a shared initiative.

Pain Point 03
No Historical Tracking

Users lacked the ability to track and review their own previous assessments to monitor progress over time.

UX Improvements & Key Features

The solution required a ground-up redesign of both the user interface and the underlying database structure to support complex, comparative psychometric evaluations.

Feature 01
Architectural Overhaul

Redesigned the database schema and user journey so that every active project is now accurately grouped under one of the six core strategic pillars, reflecting the true structure and values of the stakeholders.

Feature 02
Comparative Dashboard

Designed and engineered a dynamic dashboard that allows users to visually compare assessment scores between the community representatives and government partners on each initiative side-by-side.

Feature 03
Personalised Tracking & Accessibility

Introduced a comprehensive history feature for users to track their own assessments over time. Modernised the interface with a seamless Dark/Light mode toggle to improve accessibility and user comfort.

Technical Implementation

To ensure the application was fast, secure, and capable of handling nested relational data, a modern JavaScript stack was selected, tailored for performance and edge computing.

Stack Element 01
Frontend UI

Built using React and scaffolded with Vite for an ultra-fast development environment and highly responsive, optimised production builds.

Stack Element 02
Database

Integrated DataStax Astra DB (NoSQL) to efficiently handle the complex, nested JSON data structures required to map multiple projects to their parent strategic pillars.

Stack Element 03
DevOps & Security

Utilised Cloudflare Workers as a serverless middle layer to securely manage API secrets, preventing exposure in the client-side code while routing data efficiently.
